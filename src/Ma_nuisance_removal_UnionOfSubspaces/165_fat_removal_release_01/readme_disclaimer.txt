Please note, this package is an older version of code from Dr. Zhi-Pee Liang's 
lab that was used in the 2016 paper:

Ma C, Lam F, Johnson CL, and Liang Z-P. 
Removal of Nuisance Signals from Limited and Sparse 1H MRSI Data Using a Union-of-Subspaces Model

Magn Res Med, 75:488-497 (2016)

There is newer improved methods building on this work coming soon in his
SPICE/SPICEx package.
